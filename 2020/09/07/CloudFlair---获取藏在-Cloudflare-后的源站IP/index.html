CloudFlair - 获取藏在 Cloudflare 后的源站IP

具体原理可以查看原作者的文章
CloudFlair 开源地址：Github
这个工具是用pytohn编写的，兼容python2.7和3.9

配置

  1. 在 https://censys.io/register 注册一个账号 (FREE)

  2. 访问https://censys.io/account/api 页面，获取账号的API ID和API secret，并设置到环境变量中

  3. 克隆整个仓库

  4. 安装依赖

  5. 运行CloudFair

用法

python cloudflair.py 后面跟上需要查找的域名即可
